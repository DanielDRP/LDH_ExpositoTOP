Detailed Description

Implementación del algoritmo GRASP para el problema Team Orienteering con Ventanas de Tiempo (TOPTW).

Este algoritmo aplica un enfoque de construcción aleatoria voraz, seguido de una búsqueda local, para encontrar una solución viable a un problema de optimización.
