\doxysubsection{Detailed Description}
Clase para evaluar la solución de un problema de Team Orienteering con Ventanas de Tiempo (\doxylink{classtop_1_1_t_o_p_t_w}{TOPTW}). 

Proporciona un método para calcular el valor de la función objetivo de una solución, evaluando la distancia acumulativa entre los nodos de cada ruta generada.
