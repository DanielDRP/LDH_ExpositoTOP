\doxysubsection{Detailed Description}
Clase que representa una instancia del problema de Team Orienteering con Ventanas de Tiempo (\doxylink{classtop_1_1_t_o_p_t_w}{TOPTW}). 

Esta clase almacena la información sobre nodos, vehículos y parámetros de tiempo y distancia necesarios para resolver el problema.
