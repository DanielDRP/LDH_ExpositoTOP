\doxysubsection{Detailed Description}
Clase principal para ejecutar el algoritmo \doxylink{classtop_1_1_t_o_p_t_w}{TOPTW} en varias instancias de problemas. 

La clase inicializa una serie de archivos de instancias y los procesa mediante el algoritmo \doxylink{classtop_1_1_t_o_p_t_w}{TOPTW} utilizando las clases {\ttfamily \doxylink{classtop_1_1_t_o_p_t_w}{TOPTW}}, {\ttfamily \doxylink{classtop_1_1_t_o_p_t_w_solution}{TOPTWSolution}}, y {\ttfamily \doxylink{classtop_1_1_t_o_p_t_w_g_r_a_s_p}{TOPTWGRASP}}. Se ejecuta el método {\ttfamily GRASP} en cada instancia con diferentes parámetros.
